# Bash-Menu

## Description

This is a simple bash script that allows you to create a menu with options.

## Usage

```bash
$ ./menu.sh
```

## Options

```bash
$ ./menu.sh -h
```

## Parameters

```bash
$ ./menu.sh -p
```

## Examples

```bash
$ ./menu.sh -e
```
